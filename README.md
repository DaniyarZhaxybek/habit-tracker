# Habit Tracker

A mobile application built with Flutter for tracking daily habits.

## Features

- ✅ Create new habits
- ✅ Edit habits
- ✅ Delete habits
- ✅ Mark habits as completed
- ✅ Local storage using Hive
- ✅ Material 3 UI

## Tech Stack

- Flutter
- Dart
- Provider
- Hive
- Material 3
